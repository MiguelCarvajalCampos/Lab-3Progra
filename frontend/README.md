# Lab 3 - Programación Frontend

Este proyecto es una aplicación web desarrollada con React y Vite.

## Instalación de Dependencias

Para instalar las dependencias del proyecto, ejecuta el siguiente comando:

```bash
npm install
```

## Ejecutar la Aplicación

Para correr la aplicación en modo de desarrollo, ejecuta:

```bash
npm run dev
```

La aplicación estará disponible en tu navegador local, típicamente en `http://localhost:5173`
